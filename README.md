## 📌 Overview

The **National Assessment Academic Drive (NAAD)** is a national learning, assessment, analytics, and certification system developed by **Cadet Media Ghana**. It operates across two complementary editions:

| Edition | Focus |
|---|---|
| 🎓 **Academic Edition** | Curriculum-Based Gamified Learning aligned with the Ghana Education Service (GES) |
| 🛡️ **Safety Edition** | Emergency Preparedness & National Safety Awareness |

Both editions share a unified platform architecture — **offline-capable, gamified, curriculum-responsive, and built for nationwide deployment** — creating a complete educational engagement ecosystem that addresses academic excellence and personal safety in equal measure.

### The NAAD Learning Cycle

```
LEARN  →  PRACTICE  →  ASSESS  →  EVALUATE  →  CERTIFY
```

This cycle ensures learners not only acquire knowledge but are assessed for comprehension, application, and retention — with performance tracked continuously across sessions.

## 🛡️ About the Safety Edition

The **NAAD Safety Edition** delivers structured, multi-agency safety education to students through interactive learning, real-life scenarios, and standardized assessments.

It is developed in collaboration with the **Ghana National Fire Service (GNFS)** — which serves as the **lead coordinating institution** under the Chief Fire Officer (CFO) — and a broad coalition of national safety and security agencies.

## 🏛️ The CFO Coordination Model

A defining feature of the Safety Edition is that the **Chief Fire Officer (CFO) of GNFS** serves as the **Lead Coordinator** of all multi-agency content integration.

| Step | Action | Detail |
|---|---|---|
| 1 | CFO Issues Directive | Formal correspondence sent to all participating agencies to prepare content |
| 2 | Institutions Prepare Content | Each agency designates a focal officer and compiles officially endorsed content |
| 3 | CFO Coordinates Review | GNFS reviews all submissions against national standards |
| 4 | Content Submitted to CMG | Reviewed content formally handed over to Cadet Media Ghana |
| 5 | NAAD Integration | CMG integrates content into the Safety Edition, crediting each institution |
| 6 | CFO & Partners Endorse | Chief Fire Officer and agency heads endorse before national deployment |

## 📱 Content Portal — This Repository

The **NAAD Content Portal** is the secure internal web app used by agency officers to **submit, review, and manage quiz questions** for the NAAD Safety Edition exam cycle.

It is built as a **Progressive Web App (PWA)** — agencies can install it on their phones or computers just like a native app, with full offline support.

### Platform Features

| Feature | Detail |
|---|---|
| 🔐 Secure Login | Each agency has a unique password; agency logo reveals on correct entry |
| ✏️ Submit Questions | Module, difficulty, 4 options, correct answer, explanation |
| 📋 My Questions | View submissions filtered by Pending / Approved / Rejected |
| 🔍 Review & Approve | CMG admin approves or rejects submissions with one click |
| 📊 Agency Progress | Track Easy / Medium / Hard question counts vs 30-question target |
| 🥇 Leaderboard | Live cadet rankings from across Ghana |
| 📈 Analytics | Average scores by module and region |
| 📢 Broadcast | CMG sends notices to all agencies |
| 💾 Export Quiz | Download approved questions as HTML quiz files for pen drives |
| 📖 Guidelines | Built-in NAAD question writing standards |

### Role-Based Access

| Role | Access |
|---|---|
| **Agency Officer** | Dashboard · Submit · My Questions · Leaderboard · Analytics · Guidelines |
| **CMG Administrator** | Everything above + Review All · Agencies · Progress · Deadlines · Broadcast · Export |

## 🚀 Deployment (GitHub Pages — Free)

The PWA install feature and offline mode require a web server. The easiest free option is **GitHub Pages**.

### Step-by-step

1. Create a free account at [github.com](https://github.com)
2. Click **+** → **New repository** → name it `naad-portal` → set to **Public** → **Create**
3. Click **Add file** → **Upload files** → drag in `naad-portal.html`, `naad-sw.js`, `README.md`
4. Click **Commit changes**
5. Go to **Settings** → **Pages** → Source: `main` branch, folder `/ (root)` → **Save**
6. Your portal is live at:
   ```
   https://YOUR-USERNAME.github.io/naad-portal/naad-portal.html
   ```

Share this URL with all agency officers.

## 📖 Question Writing Standards

- Minimum **30 questions per module** — 10 Easy · 12 Medium · 8 Hard
- **Factually accurate** — based on official agency guidelines and national safety standards
- **Four plausible options** — all four answers should seem possible to a student
- **One unambiguous correct answer** — no room for interpretation
- **Strong explanation** — cite facts, laws, or official guidelines
- **Scenario-based preferred** — e.g. *"A student sees smoke — what should they do first?"*
- No trick questions, duplicates, or politically sensitive content
